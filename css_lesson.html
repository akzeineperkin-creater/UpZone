Основы стилей

CSS Флешкарты

CSS — это язык стилей. Если HTML — это кирпичи, то CSS — это краска, обои и расстановка мебели.

Как работает правило CSS:

h1 {
  color: red;
  font-size: 20px;
}

Три способа подключения:

  • Внешний файл: тег <link> (лучший способ)
  • Внутри головы: тег <style>
  • Инлайново: атрибут style="..." прямо в теге

Попробуй в VS Code

/* 1. Сделай все заголовки синими */ h1 { color: skyblue; text-align: center; } /* 2. Добавь красивую рамку для блоков */ .card { background-color: white; border-radius: 10px; box-shadow: 0 4px 8px rgba(0,0,0,0.1); padding: 20px; }
