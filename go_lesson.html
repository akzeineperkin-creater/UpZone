Go Флешкарты

Go — это как C, но без головной боли. Простой синтаксис, молниеносная компиляция и встроенная поддержка многопоточности.

Классический Hello World на Go:

package main

import "fmt"

func main() {
  fmt.Println("Hello, Go!")
}

Где используют Go:

  • Серверы и API: Gin, Echo, Fiber
  • Микросервисы: Docker, Kubernetes написаны на Go
  • CLI-утилиты: быстрые консольные приложения

💎 Творческое задание: Напиши функцию!

Напиши функцию на Go, которая принимает имя пользователя и возвращает приветствие:

  • Функция называется greet
  • Принимает параметр name string
  • Возвращает строку вида "Привет, Name!"

Эталонное решение:

func greet(name string) string {
    return "Привет, " + name + "!"
}

func main() {
    fmt.Println(greet("Алексей"))
    // Выведет: Привет, Алексей!
}
