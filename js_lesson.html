JS Флешкарты

JavaScript — это мозг сайта. Он обрабатывает клики, считает данные и делает страницу живой.

Базовый синтаксис:

let name = "Аня"; // Переменная function sayHello() { alert("Привет, " + name); }

Подключение:

Скрипт вставляется перед закрывающим тегом </body>:

<script src="script.js"></script>

Твой первый скрипт

console.log("Код работает!"); alert("Привет из JavaScript!");

✅ Код принят! Правильное решение:

const element = document.querySelector('#status'); element.textContent = "Задание выполнено!";
