HTML — это скелет любого сайта. Он говорит браузеру: «Здесь заголовок, здесь текст, здесь кнопка».

Как конструктор: теги — это кирпичики.

  • <h1>...</h1> — большой заголовок
  • <p>...</p> — абзац текста
  • <img src="..."> — картинка
  • <a href="...">...</a> — ссылка

Главное правило:

<тег> содержимое </тег>

Шаг 1 — Создай первую страницу за 2 минуты

1. Открой VS Code
2. Скопируй код ниже
3. Сохрани как hello.html
4. Открой в браузере

<!DOCTYPE html> <html lang="ru"> <head> <meta charset="UTF-8"> <title>Моя первая страница</title> </head> <body> <h1 style="color: #c084fc;">Привет, мир! 🌍</h1> <p>Я создал свой первый файл!</p> </body> </html>
