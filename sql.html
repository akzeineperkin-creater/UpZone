Базовые Команды

SQL Флешкарты

SQL — это язык запросов. Если база данных — это огромный склад, то SQL — это кладовщик, который находит нужный товар по вашему списку.

Как выглядит простой запрос:

SELECT name, age
FROM users
WHERE age > 18;

Типы команд (языки):

  • DQL: получение данных (SELECT)
  • DML: изменение данных (INSERT, UPDATE, DELETE)
  • DDL: структура таблиц (CREATE, DROP, ALTER)

Примеры запросов

/* 1. Выбрать всех сотрудников из отдела 'IT' */ SELECT * FROM employees WHERE department = 'IT'; /* 2. Добавить нового пользователя в таблицу */ INSERT INTO users (name, email) VALUES ('Ivan', 'ivan@example.com');

💎 Творческое задание: Напиши запрос!

Напиши SQL-код для таблицы products, чтобы:

  • Выбрать колонки title и price
  • Оставить только товары, где цена больше 500
  • Отсортировать результат по price

Эталонный запрос:

SELECT title, price 
FROM products 
WHERE price > 500 
ORDER BY price;
