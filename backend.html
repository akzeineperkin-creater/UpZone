Базовые Команды

Backend Флешкарты

Backend — это «кухня» приложения. Если Frontend — это красивое меню и зал, то Backend — это повара, продукты и склад данных.

Как выглядит простой JSON-ответ:

{
  "status": 200,
  "message": "Success",
  "data": [...]
}

Типы технологий:

  • Runtime: Node.js, Python, Go, Java
  • API: REST, GraphQL, gRPC
  • Databases: PostgreSQL, MongoDB, Redis

Примеры логики

/* 1. Обработка GET-запроса на получение профиля */ app.get('/api/user/:id', (req, res) => { const userId = req.params.id; return res.status(200).json({ id: userId, role: 'admin' }); }); /* 2. Middleware для проверки прав доступа */ const checkAuth = (req, res, next) => { if (!req.isAuthorized) return res.status(401).send(); next(); };

💎 Творческое задание: Напиши логику!

Напиши структуру JSON-ответа для объекта user, чтобы:

  • Указать поля username и email
  • Добавить массив roles с правами доступа
  • Показать статус isOnline (true/false)

Эталонный JSON:

{
  "username": "alex",
  "email": "alex@mail.com",
  "roles": ["admin", "editor"],
  "isOnline": true
}
