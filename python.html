Python Флешкарты

Python — это интерпретируемый язык. Его код читается почти как английский текст, что делает его идеальным для первого языка программирования.

Пример простой функции:

def greet(name):
  print(f"Привет, {name}!")

greet("Мир")

Ключевые особенности:

  • Динамическая типизация: не нужно указывать тип переменной явно.
  • Огромная библиотека: готовые решения для любых задач.
  • Отступы: структура кода определяется пробелами, а не скобками.

Попробуй в IDLE или VS Code

# 1. Списки и циклы fruits = ["яблоко", "банан", "вишня"] for fruit in fruits: print(fruit.upper()) # 2. Условные конструкции age = 20 if age >= 18: print("Доступ разрешен") else: print("Доступ закрыт")

Возможное решение:

def calculate(a, b):
    s = a + b
    if s > 100:
        return "Big"
    return s
